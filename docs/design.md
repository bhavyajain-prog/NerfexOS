# Design Language

## Theme

Blood Moon

## Philosophy

Darkness is the default.
Color is earned.

## Visual Style

- Cinematic
- Minimal
- Atmospheric
- High contrast
- Heavy use of negative space

## Primary Colors

Background : #0F0F10

Surface : #1A1A1D

Foreground : #F2F2F2

Accent : #B31217

Muted : #666666

## Typography

UI Font:
Inter

Terminal Font:
JetBrains Mono Nerd Font

Decorative Font:
TBD

## Animation

Animations should be:

- Smooth
- Slow
- Intentional

Avoid flashy effects.

## UI Rules

- No unnecessary gradients
- No rainbow color schemes
- Thin borders
- Consistent spacing
- Accent color only for interactive elements
- Wallpapers should dominate the desktop

## Inspiration

- Ghost of Tsushima
- Sekiro
- Dark Souls
- Elden Ring
- Japanese minimalism
