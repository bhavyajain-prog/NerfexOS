# Vision

## Purpose

NerfexOS aims to create a desktop environment that feels intentional from the moment the power button is pressed until the system shuts down.

Rather than focusing only on appearance, the project combines visual consistency, system optimization, reproducibility, and automation into a single experience.

## Core Principles

- Minimalism over clutter
- Consistency over customization
- Automation over manual configuration
- Performance over unnecessary effects
- Reproducibility over one-off tweaks

## Goals

- Remove unnecessary distribution branding
- Build a cohesive operating system identity
- Maintain complete configuration through version control
- Enable full system recreation from the repository

## Non Goals

- Becoming a Linux distribution
- Supporting every desktop environment
- Maximizing visual effects at the expense of usability
