# NerfexOS

> A highly customized, Arch Linux based desktop experience built around a minimal, cinematic design philosophy.

![Version](https://img.shields.io/badge/version-0.0.4-blue) ![Status](https://img.shields.io/badge/status-Planning-orange) ![Base](https://img.shields.io/badge/base-Arch%20Linux-1793D1)

## Vision

NerfexOS is not intended to be another Linux "rice".

The objective is to build a cohesive operating system experience—from power-on to shutdown—with every visual, functional, and technical component designed around a single design language.

The project prioritizes:

- Consistency
- Minimalism
- Performance
- Automation
- Reproducibility

## Design Language

Current design direction:

**`Blood Moon`**

A cinematic dark theme built around:

- Monochrome interface
- Crimson accents
- Heavy negative space
- Slow, subtle animations
- Minimal distractions

> Darkness is the default. Color is earned.
